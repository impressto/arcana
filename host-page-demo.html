Integration Instructions

To integrate the Arcana app:

  1. Build the app: npm run build
  2. Include the CSS: <link rel="stylesheet" href="dist/assets/index.css">
  3. Include the JS: <script src="dist/assets/index.js"></script>
  4. The app will automatically hide the #arcana-info-box when wizards start
